# 3 meses

Pagina estatica pronta para GitHub Pages.

## Deploy no GitHub Pages

1. Acesse `Settings` > `Pages` no repositorio.
2. Em `Build and deployment`, selecione `Deploy from a branch`.
3. Escolha o branch `main` e a pasta `/(root)`.
4. Salve. A pagina ficara em `https://gpedrosouza.github.io/3-meses/`.

O arquivo publicado na raiz e `index.html`, criado a partir de `nossos-3-meses.html`.
